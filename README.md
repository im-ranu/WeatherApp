# WeatherApp
To get weather information of your city with forecast upto 5 days.


# Modules use in project

  <ul>
  <li>Retrofit</li>
  <li>Lottie - Animations</li>
  <li>Firebase</li>
  <li>Zxing - QR Code Scanner</li>
  </ul>
